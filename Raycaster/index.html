Raw Data

This section contains the raw output produced by the scripts above. For the resolution-scaling run and the parameter-sweep performance experiment, the output consists of human- and machine-readable log files; for the quality run, it is a large collection of images. These files can be further processed with the scripts above to generate the plots shown in our paper.

  • Output of the Resolution Scaling Run logs/resolution.zip
  • Output of the Performance Run logs/performance.zip
  • Output of the Quality Run logs/images.zip Warning: about 60 GB download

Interaction Help
Mouse
Standard mode
  • Left-click + dragrotate the volume around its center (trackball-style)
  • Right-click + dragadjust the transfer function interactively (brightness/opacity shaping)
Transfer function editor mode
  • Left-click + dragedit the transfer function curve for the active channel directly
Keys
  • Number + Returnjump directly to a rendering method by index
  • Isave a screenshot and the current scene URL, then download both files
  • Otoggle orthographic and perspective projection
  • Ctoggle constant sample count (keeps true+virtual samples roughly constant)
  • V / Shift+Vcycle volumes backward / forward
  • N / Mcycle render modes backward / forward
  • Ptoggle between the current mode and the previously used mode
  • W / Qdecrease / increase the sampling rate (step size along the ray)
  • Rreset the view to the default camera/transform
  • D / Shift+Ddownsample the volume by 2 / return to the next higher resolution level
  • Uprint the current scene URL to the browser console
  • Topen the transfer function editor (enables editor mode interactions)
  • Achange the active transfer function channel while in editor mode
  • Gtoggle Gaussian vs Smoothstep transfer function behavior for right-drag interaction
  • Up / Shift+Upzoom out / move upward
  • Down / Shift+Downzoom in / move downward
  • Left / Shift+Leftdecrease virtual samples / move left
  • Right / Shift+Rightincrease virtual samples / move right
  • F1 / Shift+F1increase / decrease clipping in x
  • F2 / Shift+F2increase / decrease clipping in y
  • F3 / Shift+F3increase / decrease clipping in z
  • F4 / Shift+F4shift the clipping window in x
  • F5 / Shift+F5shift the clipping window in y
  • F6 / Shift+F6shift the clipping window in z
