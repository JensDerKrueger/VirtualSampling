# Documentation

The top-level [README](../README.md) is intended as a compact project overview.
More detailed documentation is split into focused guides in this directory:

- [Installation and Build](installation.md)
- [Scripting Language and Automation](scripting.md)
- [Usage and Controls](usage.md)

These pages are adapted from the appendices in the paper sources and updated to
match the current repository layout:

- `Raycaster/` contains the application, scripts, Xcode project, and web build
- `Raycaster/VS/Raycaster.sln` is the Windows Visual Studio solution
- `Raycaster/Scripts/` contains the example automation scripts

If you only want to try the project without compiling anything locally, start
with the hosted web version at [cgvis.de/vs](https://www.cgvis.de/vs).
